设计一个四足机器人控制APP中的“退出确认弹窗（Exit Confirmation Modal）”，用于安卓横屏界面。

--------------------------------
一、使用场景
--------------------------------
用户点击“退出”按钮后触发。

该操作属于高风险行为：
退出后将失去对机器人的控制。

--------------------------------
二、设计目标
--------------------------------
- 明确风险提示
- 防止误触
- 快速做出决策（取消 / 确认）
- 保持与主界面一致的科技感风格

--------------------------------
三、弹窗布局（居中）
--------------------------------

弹窗位于屏幕中央，背景为半透明遮罩。

尺寸：
- 横向长方形（适配横屏）
- 圆角（16–20px）

--------------------------------
四、弹窗内容结构
--------------------------------

【顶部图标】
- 警告图标（黄色 ⚠️）

【标题】
- “确认退出控制？”

【正文说明】
- “退出后将无法继续控制机器人，请确认当前环境安全。”

👉 文案要求：
- 简洁
- 强调“失去控制”风险

--------------------------------
五、按钮设计（关键）
--------------------------------

按钮布局：左右两个按钮（横向排列）

左侧按钮：
- 文案：取消
- 样式：
  - 深色背景 / 描边按钮
  - 白色文字
  - 视觉弱化

右侧按钮：
- 文案：确认退出
- 样式：
  - 红色按钮（#FF3B30）
  - 白色文字
  - 高对比（强调危险）

👉 间距适中，避免误触

--------------------------------
六、交互逻辑
--------------------------------

- 点击“取消” → 关闭弹窗
- 点击“确认退出” → 执行退出逻辑

- 点击遮罩：
  → 关闭弹窗（等同取消）

--------------------------------
七、背景处理
--------------------------------

- 主界面变暗（黑色遮罩 opacity 0.5）
- 禁止操作背景（锁定）

--------------------------------
八、视觉风格
--------------------------------

- 深色主题（黑/深灰）
- 半透明面板（磨砂玻璃效果）
- 字体白色
- 红色用于危险操作

--------------------------------
九、动效
--------------------------------

- 弹窗出现：scale + fade（150–200ms）
- 遮罩渐入
- 按钮点击有轻微缩放反馈

--------------------------------
十、设计要求
--------------------------------

- 安卓横屏适配
- 操作区域足够大（适合触控）
- 信息层级清晰（图标 → 标题 → 文案 → 按钮）
- 避免误触（按钮间距合理）
