设计一个四足机器人控制APP的“设置面板（Settings Panel）”，用于安卓横屏界面。

--------------------------------
一、设计目标
--------------------------------
- 仅补充当前主界面尚未包含的功能
- 不重复已有控制功能（如：体态、跟随、速度、控制方式等）
- 作为二级页面/侧边面板使用
- 操作简单、逻辑清晰、适合户外实时控制

--------------------------------
二、布局结构
--------------------------------

采用“左侧控制面板 + 右侧内容区”的结构：

左侧：功能控制区（约30%宽度）
右侧：相机画面 + 地图浮层

--------------------------------
三、左侧控制面板
--------------------------------

顶部：
- 分段切换（Segmented Control）
  [ 基础功能 ] [ 地图与任务 ]

--------------------------------

模块一：智能辅助（开关）

- AI辅助（开 / 关）
  - 说明：开启后提供智能控制优化

👉 不包含“姿态调整”（已在主界面提供）

--------------------------------

模块二：地图控制（开关 + 入口）

- 地图显示（开 / 关）
  - 打开后右侧显示地图

--------------------------------

模块三：任务操作（按钮）

按钮组：

- [ 自主充电 ]
- [ 自主出桩 ]

样式：
- 黑底按钮
- 白字
- 圆角
- 轻微阴影

--------------------------------

模块四：地图管理

按钮：

- [ 建图导航 ]
- [ 离线地图管理 ]

--------------------------------

模块五：系统信息

- 版本号（如：V2.0.3）

--------------------------------
四、右侧区域（动态内容区）
--------------------------------

默认状态：
- 显示实时相机画面（与主界面一致）

当“地图显示”开启：

👉 在右侧叠加地图浮层（不完全遮挡画面）

地图功能包括：

- 当前机器人位置（marker）
- 简单路径或定位信息
- 右下角缩放/全屏按钮

--------------------------------
五、交互逻辑
--------------------------------

- 点击主界面“设置按钮” → 面板从左侧滑出
- 面板宽度固定（约30%）

关闭方式：
- 点击关闭按钮
- 点击画面区域

地图逻辑：
- 打开地图开关 → 右侧立即显示地图
- 关闭 → 恢复纯相机画面

--------------------------------
六、视觉风格
--------------------------------

- 背景：深黑（#000000）
- 面板：半透明（玻璃质感）
- 字体：白色
- 强调色：
  - 绿色（正常/开启）
  - 红色（关键操作）

--------------------------------
七、设计要求
--------------------------------

- 安卓横屏适配
- 分区清晰（辅助 / 地图 / 任务 / 系统）
- 控件间距合理
- 不与主界面功能重复
- 优先保证操作效率
