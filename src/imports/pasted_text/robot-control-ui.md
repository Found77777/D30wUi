设计一个四足机器人控制APP的主界面（控制页），用于安卓横屏设备。

要求：
在保留所有功能的前提下，对按钮表达（文案）进行用户化重构，并优化整体布局与信息层级，使界面更直观、易理解。

--------------------------------
一、设计目标
--------------------------------
- 所有功能必须完整保留（不可删除）
- 允许修改按钮名称，使其符合用户直觉（禁止使用工程术语，如“A/B按钮”）
- 优化布局分区，使信息结构清晰
- 适配横屏双手操作（左右拇指控制）

--------------------------------
二、背景与整体结构
--------------------------------
- 背景：纯黑色（#000000）
- 主体：全屏实时相机画面
- UI：半透明浮层（不遮挡中心区域）

采用三层结构：
Layer 1：相机画面
Layer 2：顶部状态信息
Layer 3：控制操作区（底部 + 两侧）

--------------------------------
三、顶部状态栏（信息展示区）
--------------------------------
半透明黑色背景（约50%透明度）

【左侧】
- 电源按钮（红色）
- 退出（小图标）

【中间】
以标签形式显示：
- 摄像头状态（前/后）
- 当前运行状态（如：待机 / 运行中）
- 诊断状态
- 当前速度（如：0.0 m/s）

【右侧】
- 电量（73%）
- 电池图标（SOC）

--------------------------------
四、模式控制区（核心分组优化）
--------------------------------
将所有模式类功能整合为三组，放在顶部下方一行：

【1】移动速度（替代：低速/中速/高速）
标题：速度
按钮：
[ 慢速 ] [ 标准 ] [ 快速 ]

【2】控制方式（替代：自主/手动）
标题：控制方式
按钮：
[ 手动控制 ] [ 自动跟随 ]

【3】运动状态（替代：卸力/常规/阻尼）
标题：运动模式
按钮：
[ 轻松 ]（卸力）
[ 标准 ]（常规）
[ 稳定 ]（阻尼）

👉 每组视觉分区明显
👉 当前选中高亮（白底）

--------------------------------
五、中间区域（视觉核心区）
--------------------------------
- 保持相机画面完整
- 不放按钮
- 保留辅助线（绿色地面线）
- 中间红框弱化（透明度降低，仅作为辅助）

--------------------------------
六、主控制区（底部）
--------------------------------

【左下角】
- 左摇杆：控制移动（前/后/左/右）

【右下角】
- 右摇杆：控制方向/转向

优化：
- 半透明
- 靠边布局，避免遮挡视野

--------------------------------
七、功能按钮重构（语义优化）
--------------------------------

将原有A/B按钮改为有语义的操作按钮：

【底部中间】
- [ 站立 / 蹲下 ]（原A）
- [ 开始跟随 ]（原B）

👉 横向排列

--------------------------------

【右下角（靠边）】
- 摄像头切换按钮（前/后）

--------------------------------

【左下角（靠边）】
- 设置按钮

--------------------------------
八、功能完整性要求（必须覆盖）
--------------------------------

界面必须包含以下功能（可重命名）：

- 电源控制
- 退出APP
- 摄像头状态显示与切换
- 运行状态
- 诊断状态
- 当前速度显示
- 三档速度调节
- 手动/自动控制切换
- 力控/阻尼相关模式
- 电量显示
- 双摇杆控制
- 跟随功能
- 设置入口

--------------------------------
九、交互要求
--------------------------------

- 所有按钮点击后有视觉反馈
- 当前模式必须有清晰状态显示
- 操作逻辑直观，不需要学习成本
- 保持所有原功能逻辑一致

--------------------------------
十、视觉风格
--------------------------------

- 黑底白字，高对比
- 强调色：
  - 红色（电源/危险）
  - 绿色（正常状态）
- 风格：极简、科技感、工业风

--------------------------------
十一、设计输出要求
--------------------------------

- 高保真UI设计
- 安卓横屏比例
- 双手操作友好
- 分区清晰（状态 / 模式 / 控制）
- 不允许信息堆叠混乱
