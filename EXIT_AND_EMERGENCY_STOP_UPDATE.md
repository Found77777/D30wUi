# 退出确认弹窗 & 急停按钮更新说明

## 🎉 更新内容

### 1. ✋ 急停按钮图标更新

**位置**：右上角（电量显示和退出按钮之间）

**变更**：
- ❌ 旧图标：Power（电源图标）
- ✅ 新图标：Hand（手掌图标）

**视觉效果**：
```
红色圆形背景 + 白色手掌图标
```

**语义表达**：
- 手掌图标更直观地表达"急停"概念
- 红色背景强调紧急操作
- 与国际通用的"停止手势"一致

**代码实现**：
```tsx
<motion.button
  whileTap={{ scale: 0.9 }}
  className="p-2 rounded-full bg-red-600 hover:bg-red-700 transition-colors"
  onClick={props.onPowerToggle}
  title="急停"
>
  <Hand className="w-4 h-4 text-white" strokeWidth={2.5} />
</motion.button>
```

## 🎮 交互逻辑

### 触发流程

```
用户点击 X 按钮
  ↓
弹窗淡入 + 缩放动画（150ms）
  ↓
背景变暗（黑色遮罩 opacity: 0.5）
  ↓
等待用户选择
```

### 用户选项

**选项1：取消**
```
样式：描边按钮（白色边框）
背景：bg-black/40
文字：白色
动作：关闭弹窗，返回控制界面
```

**选项2：确认退出**
```
样式：实心按钮
背景：bg-red-600（红色警告）
文字：白色，加粗
动作：关闭弹窗 → 执行退出逻辑
```

**选项3：点击遮罩**
```
动作：关闭弹窗（等同取消）
```

## 🔒 安全设计

### 防止误触

1. **双重确认**：需要点击两次（X按钮 → 确认退出）
2. **视觉警告**：黄色警告图标 + 红色确认按钮
3. **文案提醒**：明确说明"退出后将无法继续控制"
4. **按钮间距**：左右按钮间距充足，避免误触

### 风险提示

```
"退出后将无法继续控制机器人，请确认当前环境安全。"
```

**关键词**：
- ❗ 无法继续控制
- ❗ 环境安全

## 💻 代码实现

### 组件文件
```
/src/app/components/ExitConfirmModal.tsx
```

### 使用方式

```tsx
const [isExitModalOpen, setIsExitModalOpen] = useState(false);

// 退出按钮点击
<button onClick={() => setIsExitModalOpen(true)}>
  <X />
</button>

// 弹窗组件
<ExitConfirmModal
  isOpen={isExitModalOpen}
  onClose={() => setIsExitModalOpen(false)}
  onConfirm={() => {
    setIsExitModalOpen(false);
    props.onExit?.();
  }}
/>
```
